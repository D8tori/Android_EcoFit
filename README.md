# 🌿EcoFit🌿 (Team Tori)

## 🪄서비스 소개
`환경보호`를 위한 습관 형성 안드로이드 앱 서비스
- 환경보호 실천 문화를 조성합니다.
- 기후 위기에 모두가 관심을 갖고 행동할 수 있는 시민 의식을 제고합니다.
- 기존 챌린지 어플과 달리 환경보호에 초점을 맞췄습니다.
